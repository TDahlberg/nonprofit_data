Nonprofit Data

This repository contains a 4-part .ZIP that unzips into a 300mb CSV file. This file contains records for every single IRS tax-exempt organization (Nonprofit) in the United States. This data comes straight from the IRS.

Project Wiki

For all field definitions, metadata, and possible values for every single column visit the project wiki.

Creating pages manually

To get this data, simply download the .ZIP file, and unzip the file that ends in ".001". This should unzip all four folders and create an extremely large CSV file. This will be much too large to use in Excel. Try using R, Python, or SQL.
